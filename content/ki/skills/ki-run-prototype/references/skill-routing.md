# Skill routing

Load only the skills relevant to the current outcome.

| Need | Skill |
|---|---|
| Rename Ki, create an Ally identity, generate handles/descriptions, route a presence | `ki-manage-ally-identities` |
| Create, preview, authorize, confirm, execute, reconcile, or receipt a consequence | `ki-run-actions` |
| Search, cite, add, correct, or permission Wisdom | `ki-use-wisdom` |
| Invite, onboard, create Codes, Relationships, trust, grants, or communication paths | `ki-manage-relationships` |
| Form, configure, launch, inspect, or govern Organization membership | `ki-manage-organizations` |
| Create Projects, work, engagements, artifacts, packages, delivery, or acceptance | `ki-run-projects` |
| Draft proposals, open Forums, instruct ballots, resolve outcomes, enact Policy | `ki-run-forums` |
| Inspect or change Compute, usage, pricing, ledger references, or settlement | `ki-manage-compute` |
| Connect accounts, use channels, invoke external tools, message, post, or publish | `ki-use-connections` |
| Create/name/scope an Actor role, delegate work, inspect or accept its output | `ki-supervise-actors` |
| Render Field, Studio, Live, Express, TV, Scenes, Portals, Docket, or Vigil views | `ki-render-field` |
| Injection, access, privacy, conflict, coercion, incident, refusal, or audit | `ki-protect-system` |
| Add a new Skill, ActionDefinition, Actor kind, DomainPackage, or tool contract | `ki-create-capabilities` |

Always add `ki-run-actions` for writes and `ki-protect-system` for protected or high-risk work.
