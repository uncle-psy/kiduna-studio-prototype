# Render the Field

Read [references/surface-grammar.md](references/surface-grammar.md).

## Render from state

1. Read fresh permission-filtered graph/prototype state.
2. Identify surface, viewport, input method, selected object, Organization/Project, and connection status.
3. Preserve stable Scene and object identity.
4. Render canonical objects and attach ACTIONS at their targets.
5. Use Clear, Context, or Focus HUD depth according to attention.
6. Show motion only for real state transition or arrival.
7. Present provenance, receipt, mode, and status on demand.
8. Offer a non-spatial structured account for accessibility or preference.

The Docket is a projection of sovereignty-requiring Actions, not an inbox, badge system, or score. Never gamify people, trust, contribution, responsiveness, or presence.

State disconnection as weather and preserve the last snapshot as a labeled snapshot, not a live Field.
