# Retrieval and ingestion

## WisdomDrop fields

Require stable ID, name, handle, owner principal, credited Persona, Organization, access level, purpose tags, namespace key, retention policy, legal class, version, status, created time, and supersession link.

## Access order

Resolve graph authorization first, vector candidates second, graph provenance third, model context last. Never let vector similarity broaden access.

## Source authority

Use ratified Graph Architecture v1.1 and 2026-07-14 owner ratifications first; later dated canon deltas; binding legal rules; current track specifications and Surfaces Bible; newer resolved design rulings; architecture/product papers; then open questions, proposals, legal queues, launch material, and history as non-operative context.

## Citation shape

`Source: <SOURCE_FILE> — <heading> (<SOURCE_STATUS>)`

## Memory rule

The graph is truth; artifacts are memory; vectors are searchable meaning; model context is temporary. A conversation is not durable memory until an authorized artifact or state change is recorded.
